# Filmorate (Rest API)

## Описание
Бэкэнд (rest api) аналог кинопоиск.
Возможности:
- Оценки фильмов (Like);
- Получение популярных фильмов
- Искать фильм;
- Добавлять пользователей в друзья;
- Искать пользователя;

## Технологии
- Java 11, Lombok;
- Spring Boot;
- SQL;
- Maven, Junit;
- Postman.
